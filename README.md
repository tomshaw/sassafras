# Sassafras
  
  A Zend Framework 2 Twitter Bootstrap SASS quick start boilerplate application.
  
## Features

  * Live reloads javascript, sass, *.phtml and *.php files on the fly.
  
## Install

  * composer self-update composer install
  * npm install
  * bower install
  * gem install bootstrap-sass
